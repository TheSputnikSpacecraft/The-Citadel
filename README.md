The Citadel is a Game of Thrones themed discussion forum which takes inspiration from classic message boards and modern platforms like Reddit and 4chan. The project focuses on delivering a functional, well-structured forum with a strong thematic identity, built using React + TypeScript on the frontend and Go on the backend, communicating via RESTful APIs.
